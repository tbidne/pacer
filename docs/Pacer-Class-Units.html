baseFactor :: Fromℤ b => a -> b Source #

Retrieve the factor relative to the base unit i.e. for some quantity x and unit u,

  x u === x * (baseFactor u) base

For instance, baseFactor Kilometer === 1_000, since

  x kilometers === x * 1_000 meters
