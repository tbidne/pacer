Pace

data Pace (d :: DistanceUnit) a where Source #

Represents a duration per distance e.g. 4'30" per kilometer. We only allow the distance to be kilometers or miles. Meters are disallowed as they chance for mistakes are high (i.e. almost all paces per meter are zero since they are so low).
